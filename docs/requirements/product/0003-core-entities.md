# Product Requirement 0003

## Purpose

This document captures the current core product nouns.

## Core Entities

### `Contributor`

A trusted human participant in the collective.

### `Soul`

A durable continuity and personality service.

Current working model:

- a standard chat endpoint
- consulted by a pilot at some interval

### `Caste`

A reusable role template:

- skills
- tools
- doctrine
- permissions
- model class

### `Character`

`Soul + Caste`

The recognizable operative identity.

### `Agent Instance`

A live embodiment:

- `Character + Compute`
- or `Caste + Compute` for thin soulless work

### `Mana Server`

A standardized compute contribution appliance:

- `LiteLLM + vLLM + collective glue`

### `Team`

A contributor-managed aggregate of:

- souls
- castes
- characters
- pilot licenses
- compute allocations

## Naming Intent

These nouns should remain stable unless there is a strong reason to change them.

They are the current product vocabulary anchors for future design and implementation work.
