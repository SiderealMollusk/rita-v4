# 0004 - Nextcloud As Front Office

## Role

Nextcloud is the human-facing office.

It is where humans and agents visibly conduct paperwork and coordination.

## Primary Affordances

- `Collectives`: durable knowledge and briefs
- `Deck`: visible work boards
- `Talk`: negotiation, handoffs, completion chatter
- `Tasks`: explicit action items
- `Notes`: scratch and short outputs
- `Calendar`: deadlines and coordination
- `Contacts`: people and collaboration metadata

## Design Principle

Agents should feel like diligent office workers:

- moving cards
- writing briefs
- leaving notes
- discussing work in Talk
- visibly progressing tasks

## V1 Tenancy

- one shared Nextcloud instance
- multiple groups inside it
- no strict multi-tenant isolation in v1
