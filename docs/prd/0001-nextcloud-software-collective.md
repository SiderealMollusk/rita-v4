# 0001 - Nextcloud Software Collective

## Thesis

The platform is a software collective:

- humans contribute direction, compute, and oversight
- agents act like visible clerical coworkers
- Nextcloud is the shared office
- the Hive is the machine core

The platform should feel socially legible, not like an opaque automation dashboard.

## Core Product Goal

Create a shared workspace where:

- contributors can direct work without micromanaging low-level execution
- soulful manager-level agents coordinate thin worker agents
- work leaves readable traces in Deck, Collectives, Talk, Notes, and Tasks
- contributors can bring compute and pilots into the collective

## Non-Goal

The product is not:

- a pure zero-trust agent sandbox
- a generic Kubernetes control panel
- a pure chatbot playground
- a purely internal machine-only orchestration system

## Primary Surfaces

- `Nextcloud`: front office, human-facing workspace
- `Hive`: soulless machine core
- `Contributor backstage`: control plane for people supplying pilots and compute
- `Mana Server`: standardized compute contribution appliance

## V1 Shape

- one shared Nextcloud instance
- group-based organization inside Nextcloud
- trusted-contributor social model
- visible work movement and accountability
- thin but real execution substrate behind the workspace
