# 0009 - Work Lifecycle And Prioritization

## Human Control Plane

Deck is the main human control plane for prioritization.

Humans should be able to:

- reorder work
- defer a plan
- abandon a plan
- elevate a plan
- move something back to planning

## Agent Behavior

Agents should:

- read Nextcloud state
- do work
- update paperwork
- make movement socially visible

## Talk

Talk is the negotiation and handoff surface.

It is useful for:

- responsibility boundaries
- completion signaling
- water-cooler accountability

## Collectives

Collectives hold:

- durable rationale
- project memory
- longer-lived planning artifacts
