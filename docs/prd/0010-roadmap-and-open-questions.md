# 0010 - Roadmap And Open Questions

## Immediate Product Direction

- solidify Nextcloud as the software collective front office
- define contributor backstage concepts
- define Hive / Mana Server interaction model
- define agent-visible paperwork behavior

## Deferred

- strict multi-tenant Nextcloud instances
- hard internal zero-trust isolation
- complex vector/index infrastructure
- advanced soul management beyond a standard chat endpoint contract

## Open Questions

- how much authority should Talk have as a machine-readable surface?
- what exact controls belong in the contributor page vs Nextcloud?
- when should a pilot become durable enough to deserve a Soul?
- how much of budget policy lives on the contributor server vs the mana server?
